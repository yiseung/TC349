GroupWork for Google Chrome

A powerful group task management tool developed by Team TechSmith in CSE 498 at Michigan State University

GroupWork for Google Chrome is a powerful group task management application developed using HTML5, CSS3, and JavaScript(AngularJS). GroupWork utilizes various API from Google such as Google Drive, Google Calendar, Google Plus, Identity, and more. Users will collaborate by creating groups, creating projects with different tasks, and assigning people to specific tasks. Users can comment on each task, attach a file and share with everyone assigned to the project, set deadlines for the task, etc.

CSE 480 Database Systems

Basic PHP program to access a database, create new queries, delete existing queries. Quick demonstration of login feature.

This was part of a project in CSE 480: Database Systems. This program demonstrates basic functionalities of accessing a user database via PHP. User can simply set up a username and and password. This will add a query to the database. Before adding the query, the app will check if there is any duplicate username in the database. If a duplicate is detected, it will alert the user.

Once a user has signed up, the user can sign in with the username and password he/she created.

There are two types of users, Admin and Student. There is admin settings page which only an admin can access, and a student settings page where both an admin and a student can access.

On Student Settings page, a user can change his/her password by typing the old password and a desired new password.

On Admin Settings page, an admin can reset the password of all student accounts. An admin can also look up anyone's information with a username. If the field is left blank, the app will query all user's information.

An admin has ability to change any user's account information. The admin can change a user's username, password, and/or user type.
